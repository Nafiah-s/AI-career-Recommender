# 🎯 AI Career Recommendation System

---

## 📌 Overview

This project is a **Streamlit-based AI Career Recommendation System** that suggests suitable career paths, job roles, and learning resources based on a user’s skills, interests, and preferences.

It helps students, freshers, and job seekers make **data-driven career decisions** using intelligent matching techniques.

---

## 🎯 Objectives

* Provide personalized career recommendations
* Match user skills with industry-relevant roles
* Identify skill gaps and suggest improvements
* Offer guided learning paths through course recommendations
* Build an interactive AI-powered web application

---

## ✨ Features

* 🎯 Skill-based career recommendation
* 🔍 Interest-based matching
* 📊 Match score using AI similarity
* 🧠 Smart skill gap analysis
* 💼 Suggested job roles
* 📚 Recommended courses
* 🌐 Interactive Streamlit UI

---

## 🛠️ Tech Stack

  * Pandas

## 📊 How It Works

1. User selects skills and enters interests
2. System processes input using TF-IDF and similarity matching
3. Matches profile with predefined career dataset
4. Displays top career recommendations with scores
